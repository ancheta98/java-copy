Javascript Basics

JavaScript is a cross-platform, object-oriented scripting language. JavaScript is extremely popular for a variety of resons. It is a small and lightweight language alowing macimum flexibility for developers to take it in a bunch of different directions. JavaScript lives inside a host environment (a web or Node server), it can be connected to the objects of these environments to provide programmatic control over them.

  • Variable DeclarationJavaScript variables are containers for storing data values-imagine a cup you fill with coffee, the cup holds the coffee, a variable holds a value. All JavaScript variables must be identified with unique names. These unique names are called identifiers. var x;
  • Variable Assignment Assignment operators assign values to JavaScript variables-our cup can now have coffee poured in it, giving our variable a value to hold. The = assignment operator assigns a vlue to a variable. var x = 10;
  • Data Types Data types are an important concept; to be able to operate on variables you need to know the data type. There are six data types that are JavaScript primitives: Boolean-true or false; null aka nothing; Number-42 or 3,14159; String-"Coding Dojo Rocks!";Array-[1,'Coding',2,'Dojo']; and Object-{first_name:'Jane', last_name:'Doe'}
